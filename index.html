Subtitles Web Site

Legendas Oficial is a website that automatically transcribes spoken video and inserts subtitles using the best YouTube font. Subscriptions are available for a day ($3.90) or a month ($9.90).

The site leverages audio enhancement techniques, Whisper for transcription, NumPy for data processing, MoviePy for video editing, and Flask for web application deployment. It efficiently processes videos in approximately 4 minutes and provides an automatically downloaded, subtitled video for the user.

IA Snake Game

Game based on AI decision-making. I used PyTorch for the AI to learn with each round. The game state is passed as calculation data for the model to calculate the functions.

I used an artificial neural network with 15 neurons in the input layer, 512 neurons in the hidden layer, and 3 neurons in the output layer. I also used ReLU activation function, Adam optimizer, and MSELoss loss function.

App Market

Software entirely created in Python that simulates the real functioning of a marketplace application. I used Tkinter to create the graphical interface and MySQL for customer registration.

It simulates all the functionalities such as registration screen, login screen, viewing market products, and even cost-benefit calculation with AI. This code will be used as a prototype for the development of an application with React Native.
